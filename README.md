# PCM Web App (V1)

A simple web application that helps users to solve PCM numerical problems and perform unit conversions.

This is the **first version (V1)** of the project. Right now it only includes the **Home** page and **Math** page. In future updates I will add **Physics** and **Chemistry** numerical solvers with more calculators, formulas, and useful tools.

This project is mainly built for learning and improving my web development skills.

---

## Features

- Simple and clean Home page
-  numerical solution
-  Unit conversion
